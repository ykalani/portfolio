# Retro Windows Portfolio

Small static portfolio app with a Windows 95/98-inspired desktop shell.

Read:
- `0to1.md` for architecture, deployment, and model selection
- `IN-PROGRESS.md` for current work

Structure:
- `index.html` app entry
- `styles.css` theme and layout
- `content.js` structured portfolio data
- `app.js` UI state and interactions
- `generator.js` launcher manifest generator and master prompt
- `app-generator-prompt.md` backend prompt contract
- `server.js` local static server
- `package.json` boot scripts
- `assets/kenney/` real UI assets and fonts
