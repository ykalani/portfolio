# Product

## Register

brand

## Users
Recruiters, engineering managers, and other developers looking to evaluate technical skills and design capabilities. They want to see real-world experience, clean code, and interactive proof-of-concept features quickly and memorably.

## Product Purpose
Showcase portfolio content (projects, experience, resume, contact details) inside an interactive, retro Windows-inspired desktop shell. It includes a live app generation playground ("App Forge") that demonstrates integration of generative AI inside frontend interfaces.

## Brand Personality
Playful, tech-nostalgic, tactile, high-craft.

## Anti-references
- Clean but generic modern landing pages (SaaS template style).
- Over-designed, heavy, slow WebGL portfolio sites.
- Flat, non-interactive PDF-style online resumes.

## Design Principles
- **Nostalgia with discipline**: The desktop UI elements feel retro and nostalgic, but performant and fully responsive.
- **Show, don't tell**: Emphasize interactive features (like draggable windows and dynamic app creation) to prove development skill.
- **Content is king**: The retro shell serves the text, links, and code, not the other way around. Keep fonts readable and spacing clean.

## Accessibility & Inclusion
- Core keyboard navigation for window management (close, minimize, focus).
- Visible focus rings and clear focus indicators for all interactive buttons.
- Respect user system preferences for reduced motion (e.g. disable window scaling animations).
